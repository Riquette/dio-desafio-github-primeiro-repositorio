# LÓGICA DE PROGRAMAÇÃO ESSENCIAL #

Programar = resolver problemas

Lógica = coêrencia de raciocínio, de ideias.
Sequencia coerente, regular e necessária de acontecimentos,coisas.

Metacongnição = pensar como você pensa

Abstração = é a habilidade de concentrar nos aspectos essenciais de
um contexto qualquer, ignorando caracteristicas menos importantes
ou acidentais.

Algoritmo = é uma sequencia de passos que resolve um problema.
 
Pseudocódigo = é uma forma genérica de escrever um algoritmo, utilizando 
uma linguagem simples.

Fluxograma = é uma ferramenta utilizada para representar graficamento o algoritmo,
isto é, a sequência lógica e coerente do fluxo de dados.
-início
-entrada
-processamento
-saída
-fim
Variáveis e constantes = na programação, uma variável é um objeto
capaz de reter e representar um valor ou expressão.
- Uma variável é um espaço na memória do coputador destinado a um dado que é alterado
durante a execução do algoritmo.

Tipos de variáveis
-Numéricas
-Caracteres
-Alfanuméricas
-Lógicas

Constates são valores imutáveis e não são alterados durante a vida úti
do programa.

Expressões Aritméticas = são expressões que utilizam operadoes aritméticos
e funções aritméticas envolvendo constantes e variáveis.

Expressões literais = são expressões com constantes e/ou variáveis que tem
como resultado valores literais. Iremos ultilizar expressões literais
na atribuição de valor para uma variável ou constante.

Tomada de decisão = quando escrevemos programas, geralmente ocorre a necessidade de decidir
o que fazer dependendo de alguma condição encontrada a execução.

Concatenação = é um termo usado em computação para designar a operação de unir o conteúdo
de duas string(Sequência de caracteres)

Estruturas de repetição = é uma estrutura que permite executar mais de uma vez o mesmo
comando ou conjunto de comandos, de acordo com uma condição ou com um 
contador.

Linguagem de programação = é uma linguagem escrita e formal que especifica um conjunto de 
instruções e regras usadas para gerar programas(software). Um software
pode er desenvolvido para rodar em um computador, dispositivo móvel ou
em qualquer equipamento que permita execução.
-alto nivel = linguem proxima da usada
-baixo nivel= linguagem de computador
-compiladas = utiliza compilador
-interpretadas= executada por um programa

Portugol? pseudo códigos.
pseudolinguagem que permite ao leitor desenvolver algoritimos estruturados
em português de forma simples e intuitiva,independentemente de linguagem de 
programação.

Desvio condicional = é utilizada a palavra reservada SE, a condição a ser testada
entre parenteses e as intruções que devem ser executadas entre chaves caso
o desvio seja VERDADEIRO.

Matriz é uma coleção de variáveis de mesmo tipo,acessíveis com um único nome e 
armazenados contiguamente na memória.

A individualização de cada variável de um vetor é feita através do uso de índices
Vetores = são matrizes de uma só dimensão.
